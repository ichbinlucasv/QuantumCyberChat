# Security Policy

## Supported Versions

We support the latest release (v1.0.0 as of December 09, 2025).

## Reporting a Vulnerability

If you discover a vulnerability:
- Do not open a public issue.
- Email ichbinlucasv@example.com with details (or use private GitHub issue if enabled).
- Expect a response within 48 hours.
- We'll credit you in the fix unless you prefer anonymity.

We use Dependabot for dependency scans and welcome security PRs.

For questions, open a non-sensitive issue.

# Contributing to QuantumCyberChat

Thank you for your interest in contributing! We welcome contributions to make this project better.

## How to Contribute

1. Fork the repo and create a feature branch (e.g., `git checkout -b feature/add-new-feature`).
2. Commit your changes with descriptive messages (e.g., `feat: add QR scanning support`).
3. Push to your fork and open a Pull Request.
4. Ensure code passes tests and follows style (use Haskell's ormolu, Python's black).

## Code of Conduct

See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

## Issues
